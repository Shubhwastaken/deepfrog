# 🧠 Customs Brain

AI-powered multi-agent customs document processing system.

## Quick Start
```bash
cp .env.example .env
# Add API keys to .env
docker-compose up --build
```

- Frontend: http://localhost:3000
- API: http://localhost:8000/docs
